# Badge


Displays a badge or a component that looks like a badge.

## Primary

```dart
ShadBadge(
  child: const Text('Primary'),
)
```

## Secondary

```dart
ShadBadge.secondary(
  child: const Text('Secondary'),
)
```

## Destructive

```dart
ShadBadge.destructive(
  child: const Text('Destructive'),
)
```

## Outline

```dart
ShadBadge.outline(
  child: const Text('Outline'),
)
```
